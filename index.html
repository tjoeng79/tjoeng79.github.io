Types of Application

  1. Desktop
  2. Web
  3. Mobile

Desktop

A desktop application is a standalone software program installed directly on a computer (Windows, macOS, Linux) that operates independently of a web browser, often offering high performance, offline functionality, and direct hardware access. These applications are used for tasks requiring significant resources, such as video editing, specialized software, or productivity tools

Common examples: Microsoft Office, Adobe Photoshop.


Web

A web application is interactive software that runs in a web browser, allowing users to perform tasks like online banking, shopping, or editing documents without installing software. Unlike static websites, web apps use server-side technologies to provide dynamic, personalized content, and they are accessed over the internet.

Common examples: Google Docs, Microsoft 365.


Mobile

A mobile application (app) is a software program designed specifically to run on mobile devices like smartphones and tablets. Unlike desktop software, these apps offer specialized, streamlined services—such as banking, social media, or gaming—and are generally downloaded from digital distribution platforms like the Apple App Store or Google Play Store.

Common examples: WhatsApp, Spotify.
